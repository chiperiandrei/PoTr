1. Cum utilizăm API-ul?

    REST API-ul disponibil are dezvoltate urmatoarele ENDPOINT-uri

  •          GET request
      /api/users - listează toți utilizatorii aplicației ( doar datele ce nu pot fi utilizate pentru logare)
      /api/poems - listează toate poemele din baza de date a aplicației
      /api/authors - listează toți autorii din baza de date a aplicației
  •          POST request
      /api/poems - inserează poeme în cadrul aplicației prin intermediul JSON-ului dat ca body.
      Exemplu de JSON  [ { "TITLE": "Example of title", "AUTHOR_ID": "1", "LANGUAGE": "RO", "STAFF_ID": "1" } ]
      /api/authors - inserează autori în cadrul aplicației prin intermediul JSON-ului dat ca body.
      Exemplu de JSON  [ { "NAME": "Name of the author", "BIRTH_DATE": "1234", "DEATH_DATE": "4321" } ]
  •          PUT request
      /api/users/{id} - actualizează numele și prenumele unui utilizator prin intermediul id-ului dat ca parametru și al JSON-ului dat ca body
      Exemplu de JSON  [ { "FIRST_NAME": "Nume1", "LAST_NAME": "Nume2" } ]
      Pot lipsi câmpuri, dar nu pot fi mai multe decât cele două menționate. ( FIRST_NAME, LAST_NAME)
      /api/poems/{id} - actualizează datele unui poem prin intermediul id-ului dat ca parametru și al JSON-ului dat ca body
      Exemplu de JSON  [ { "TITLE": "Liceu", "LANGUAGE": "RO", "ID_AUTHOR": "1", "ID_STAFF": "2" } ]
      Pot lipsi câmpuri, dar nu pot fi mai multe decât cele două menționate. ( TITLE, LANGUAGE,ID_AUTHOR,ID_STAFF)
      /api/authors/{id} - actualizează datele unui autor prin intermediul id-ului dat ca parametru și al JSON-ului dat ca body
      Exemplu de JSON  [ { "NAME": "Vasile Alessaacsandri", "BIRTH_DATE": "1821", "DEATH_DATE": "1890" } ]
      Pot lipsi câmpuri, dar nu pot fi mai multe decât cele două menționate. ( NAME, BIRTH_DATE,DEATH_DATE)
  •          DELETE request
      /api/users/{id} - șterge utilizatorul dat prin parametrul {id}
      /api/poems/{id} - șterge poemul dat prin parametrul {id}
      /api/authors/{id} - șterge autorul dat prin parametrul {id}
